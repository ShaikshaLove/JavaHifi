Converting a simple maven project into spring boot project

Converting a simple maven java project into spring boot project is easy and simple. we only make our maven project as the child project to spring-boot-starter-parent-project.

  • In step-2, we just made our project as child, so all default behavior could be inherited into our project
  • Spring boot does 80% of the configuration to run spring application and we only have to configure 20% the configuration
  • application.properties file, which is at class path,allows us to modify the default configuration
  • Spring boot configures required configuration to run spring applications by looking available jars in class path.
  • You don't need to provide version in step-3,the version will be picked from parent project by maven build tool to download the required jars from remote repository
  • @EnableAutoConfiguration annotation adds auto configuration feature to our application.
  • When spring boot finds the auto configuration is to be enabled,Spring boot looks into every jar which we added in the classpath to get spring.factories file
  • spring.factories file have the auto-configuration details. Spring boot configures the coniguration on the basis of @Conditional and @Configuration annotations
  • @ComponentScan does add component scan feature to the package where StartApp.java file is located. interestingly it's sub packages also gets scanned
  • All the beans which are annotated with Stereo type annotations gets identified and will be registered in th spring container.
  • The classes,which are annotated with stereo type annotations,must be placed in the package or in the subpackage of the package where the StartpApp.java file is located
  • here,@Configuration gives a chance to do extra configuration with @Bean annotation
