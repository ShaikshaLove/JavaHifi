Creating a rest resource

Creating a rest service with spring boot is quite easy . After getting a simple spring boot project ,we only have to create rest controller which does the actual work such as serving rest services



  • here, @RestController is a stereotype annotation.The spring bean,which is annotated with this annotation,gets identified by spring container that registers the spring bean in the spring container
  • @RestController is formed with @Controller and @ResponseBody
  • @ResponseBody annotation tells the spring container that write the return result of handler method to the body of the response document. spring container uses the value of the produces attribute of mapping annotations to select appropriate message converter to convert the return result of handler method to specific format such as json/xml/string etc. this result is added to the body of the response document
  • Coming to @GetMapping is a annotation and It is used to map a URL pattern to a spring controller's handler method.When the spring container finds the incoming request's URL pattern along with request method is matched with any these mapping annotion's URL pattern and request method, spring container calls that handler method to handle the incoming request
  • @RequestBody annotation tells the spring container to bind the incoming request payload into pojo class object and provide that object as parameter to handler method. in the above example Weather is pojo class.
  • @PathVariable annotation injects the value of template variable to handler method.In the above example the parameter name must match with template variable name otherwise spring container wont inject the value. we can also use the annotation like this (@PathVariable("weatherLocationId") String id)
