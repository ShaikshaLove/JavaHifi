Spring boot introduction

Spring boot is a tool and has variety range of starter projects that allows to build different projects in a simple manner.


Problems with Spring?
  1. Huge framework
  2. Multiple setup steps
  3. Multiple configuration steps
  4. Multiple build and deploy steps
Can we abstract Spring's problems?

Yes! we can with help of spring boot. some features mentioned below

  • Opinionated:- Spring boot does do certain configuration to reduce burden on developers. spring boot does allow us to change the default configuration
  • Stand-alone:- We no need to build war file to run the application on server. We can run the application as stand-alone because of this multiple builds would be reduced
  • Embedded servers:- Spring boot comes with three embedded servers which are tomcat,jetty and undertow
